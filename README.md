# Atlanta BitDevs

[![Netlify Status](https://api.netlify.com/api/v1/badges/3dabcadd-d728-4a01-b01e-b7a556e40c29/deploy-status)](https://app.netlify.com/sites/bitdevs-upgrade/deploys)

The home of the [Atlanta BitDevs](https://atlbit.dev) meetup — a place for free and open Bitcoin education in Atlanta.

Discussions are technical but open to people of all skill levels. Join the community and **#LearnBitcoinTogether**!

## Getting Started

This is a [Next.js](https://nextjs.org/) project.

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see the site.

## Adding Events

Event pages live in `content/events/` as Markdown files with the naming convention `YYYY-MM-DD-event-name.md`. The site picks these up automatically.
