# Atlanta BitDevs

[![Netlify Status](https://api.netlify.com/api/v1/badges/3dabcadd-d728-4a01-b01e-b7a556e40c29/deploy-status)](https://app.netlify.com/sites/bitdevs-upgrade/deploys)

The home of the [Atlanta BitDevs](https://atlbit.dev) meetup — a monthly Socratic Seminar exploring the latest developments in Bitcoin, Lightning, and related technology.

## Getting Started

This is a [Next.js](https://nextjs.org/) project.

```bash
# Install dependencies
pnpm install

# Run the development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally.

## Content

Meetup events live in the `content/events` directory as Markdown files. Each file is named with a date prefix (`yyyy-mm-dd-event-name.md`).
